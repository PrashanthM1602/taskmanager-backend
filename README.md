## 📌 Overview

**Productivity Mind** is an AI-driven workflow system designed to transform unstructured PDF content into structured tasks.

It leverages **LLM pipelines, GraphQL APIs, and modern backend architecture** to automate task creation, scheduling, and tracking.

✔ Converts raw documents into actionable workflows  
✔ Reduces manual effort through automation  
✔ Provides analytics for productivity tracking  

---

## ✨ Key Features

- 📄 **PDF Upload & Analysis** – Extracts content from documents  
- 🤖 **AI Task Generation** – Converts content into structured tasks  
- 🧩 **Hierarchical Task System** – Parent & subtask modeling  
- 📊 **Productivity Analytics** – Track completion & delays  
- 🔐 **JWT Authentication** – Secure user access  
- ⚡ **GraphQL API** – Efficient and scalable data fetching
